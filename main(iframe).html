HTML이란?

Hyper Text Markup Language의 약자로
웹 문서를 만들기 위해 사용하는 기본적인 웹 언어의 한 종류이다. 인터넷에 있는 대부분의 웹 페이지들은 HTML으로 작성된다.
<head>, <body>, <h1>, <p> 등의 요소로 구성되어 있으며, HTML 명령어를 이용해 문서의 글자의 크기, 글자의 색, 문서의 이동(하이퍼링크)등을 지정해줄 수 있다. 우리는 이를 이용해서 웹페이지를 만들 수 있다.

HTML 문서의 예


<!DOCTYPE html>
<html>
<head>
<title>타이틀</title>
</head>
<body>

<h1>제목입니다.</h1>
<p>내용입니다.</p>

</body>
</html>
