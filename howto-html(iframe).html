HTML 구성요소 알아보기


<!DOCTYPE html>
-> HTML문서임을 선언한다

<html>
-> html문 시작

<head>
-> head문 시작

<title>상단 타이틀</title>
-> 문서의 제목을 지정

</head>
-> head문 끝


<body>
-> body문 시작


<h1>제목입니다.</h1>
-> h 태그

<p>내용입니다.</p>
-> p 태그


</body>
-> body문 끝

</html>
-> html문 끝


  • <!DOCTYPE html>: 이 문서는 HTML5로 작성되었음을 선언한다.
  • <html>: HTML 페이지의 뿌리요소
  • <head>: 페이지의 메타데이터를 포함
  • <title>: 문서의 제목을 지정 (브라우저의 최상단에 표기되어있음)
