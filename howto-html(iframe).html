HTML 구성요소 알아보기


<!DOCTYPE html>
-> HTML문서임을 선언한다

<html>
-> html문 시작

<head>
-> head문 시작

<title>저의 홈페이지 입니다.</title>
-> 문서의 제목을 지정

</head>
-> head문 끝


<body>
-> body문 시작


<h1>제목입니다.</h1>
-> h1 태그

<p>내용입니다.</p>
-> p 태그


</body>
-> body문 끝

</html>
-> html문 끝


  • <!DOCTYPE html>: 이 문서는 HTML5로 작성되었음을 선언한다.
  • <html>: HTML 페이지의 뿌리요소
  • <head>: 웹문서의 메타데이터를 포함 (글꼴정보, 글의 스타일, CSS 등)
  • <title>: 웹문서의 제목을 지정 (브라우저의 최상단에 표기되어있음)
  • <body>: 실제로 우리가 볼 수 있는 웹문서의 내용을 포함
  • <h1>: 문서 각 단락의 제목 (h옆의 숫자가 작을수록 중요도 UP!)
  • <p>: 문서 각 단락의 내용

  • ✶HTML 태그는ㅤ<태그종류>내용</태그종류>ㅤ의 형식으로 입력한다! 잊지말자!

하이퍼링크 속성

하이퍼링크 속성은 <a>태그로 정의된다.

<a href="http://home.chungwoon.ac.kr/home/index.do">청운대학교 바로가기</a>

청운대학교 바로가기

또한 하이퍼링크 속성에 덧붙여 타겟 속성이라는 것이 있다.
타겟 속성은 하이퍼링크로 연결된 문서가 어떤 형식으로 열릴지 정하는 역할을 한다.

  • _blank: 새 창으로 문서가 열리게 한다.
  • _html: 현재 그 창에서 열리게 한다.
  • _parent: 부모 창에서 열리게 한다.
  • _top: 문서의 상위 창에서 열리게 한다.


이미지 속성

이미지 속성은 <img>태그로 정의된다.

<img src="사과.png">

또한 이미지 속성에 다양한 속성들을 추가로 넣을 수 있다.

<img src="사과.png" alt="사과" style="width:300px; height:300px;">


여기서 alt 속성은 이미지를 대신해서 나올 텍스트를 보여주는 역할을 한다.
예를 들어 인터넷 문제나 서버 오류로 alt 텍스트가 'nice car'인 이미지를 불러오지 못했을 때, 엑스박스 표시 옆에 'nice car'이라는 텍스트가 나오게 된다.
그 외에도 시각장애인용 페이지를 만들거나 (이미지의 alt 텍스트를 읽어줌), 다른 사람이 코드를 수정하거나 열람할 때 이게 무슨 이미지인지 알려주는 등 여러 역할을 하므로 alt 속성이 없어도 작동은 하지만 되도록이면 꼭 넣어주도록 하자.
width와 height는 이미지의 넓이와 높이를 지정한다. '사과.png'를 넓이 300픽셀, 높이 300픽셀로 나타내었다.

스타일 속성

우리는 HTML 요소에 스타일 속성을 넣을 수 있다. 스타일 속성은 HTML 요소에 글자의 크기, 폰트, 배경의 색 등을 설정하는 역할을 한다.
기본 형식은 다음과 같다.

<태그 style="속성:값;">

스타일 속성에는 다음과 같은 것들이 있다. 여러 개 넣는것도 가능하다.

  • background-color: 배경의 색을 지정한다.
  • color: 글자의 색을 지정한다.
  • font-family: 글꼴을 지정한다.
  • font-size: 글자의 크기를 지정한다.
  • text-align: 글자의 정렬 상태를 지정한다.
<h1 style="font-family:verdana; background-color:green;">Delicious apple!</h1>
-> h1 요소의 폰트를 'verdana'로 설정하고 글자 배경색을 'green'으로 한다.

Delicious apple!

<p style="font-family:lucida console; font-size:200%; color:darkred; text-align:center;">;Do you love an apple?</p>
-> p 요소의 폰트를 'lucida console'로 설정하고 글자 크기를 200%로 한다. 글자 색은 'darkred', 그리고 가운데 정렬한다.

Do you love an apple?
