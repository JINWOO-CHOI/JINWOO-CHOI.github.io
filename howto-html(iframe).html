HTML 구성요소 알아보기


<!DOCTYPE html>
-> HTML문서임을 선언한다

<html>
-> html문 시작

<head>
-> head문 시작

<title>저의 홈페이지 입니다.</title>
-> 문서의 제목을 지정

</head>
-> head문 끝


<body>
-> body문 시작


<h1>제목입니다.</h1>
-> h1 태그

<p>내용입니다.</p>
-> p 태그


</body>
-> body문 끝

</html>
-> html문 끝


  • <!DOCTYPE html>: 이 문서는 HTML5로 작성되었음을 선언한다.
  • <html>: HTML 페이지의 뿌리요소
  • <head>: 웹문서의 메타데이터를 포함 (글꼴정보, 글의 스타일, CSS 등)
  • <title>: 웹문서의 제목을 지정 (브라우저의 최상단에 표기되어있음)
  • <body>: 실제로 우리가 볼 수 있는 웹문서의 내용을 포함
  • <h1>: 문서 각 단락의 제목 (h옆의 숫자가 작을수록 중요도 UP!)
  • <p>: 문서 각 단락의 내용

  • ✶HTML 태그는ㅤ<태그종류>내용</태그종류>ㅤ의 형식으로 입력한다!
